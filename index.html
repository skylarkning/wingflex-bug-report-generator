Check for an existing report first

Before filing, search the existing issues in the wingflexsim/Bridge-Software repository for the same or a closely related problem.

Do not file the issue again if a duplicate or existing report is found. Duplicate reports slow down the team's workflow. Add any useful new details to the existing issue instead.
